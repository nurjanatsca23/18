Что это?

Это система, которая помогает правильно работать с кодом в команде.

Как работает?

Kонтролировать выполнение,задач позволяeт упорядочить работу сотрудников, работает над автоматизацией операций.

Зачем это нужно?

Это помогает отслеживать, кто и что изменил в прокте, защищает от случайных ошибок, и позволяет легко возвращаться к старым версиям кода.

Что такое git rebase?

Процесс перемещения или объединения последовательности коммитов в новый базовый коммит.

Отличия от git merge

git rebase создает новый набор коммитов, применяемых поверх целевой ветки, а git merge создает новый коммит слияния, который объединяет изменения из обеих ветвей.

Что такое git fetch

Команда Git Fetch используется для извлечения всех изменений из удаленного репозитория в локальный репозиторий. Он не вносит никаких изменений в текущий рабочий каталог. Он хранит все изменения в отдельной ветке, называемой веткой удаленного отслеживания.
